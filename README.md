# PDF Q&A Bot (Cloud-Based)

This project is a **cloud-based PDF Q&A bot** using:

- **LangChain** for RAG pipeline
- **FastAPI** backend
- **Gradio** frontend
- **Chroma** vector store
- **OpenAI / OpenRouter** LLM

---

## Features

- Upload a PDF file
- Ask questions based on the PDF content
- Fully cloud-based: backend hosted on Render/Railway/Cloudflare
- Frontend on Gradio or HuggingFace Spaces
